git clone - клонирование удаленного репозитория 
git checkout -b name_branch - создание новой ветки и переход к ней
git branch - просмотр всех веток
git status - просмотр измененных файлов 
git add . - индексация файлов
git commit -m "commit text" - фиксация изменений с кратким сообщением
git push - отправка фиксаций в удаленный репозиторий
git pull - получение фиксаций из удаленного репозитория
git merge name_branch - слияние веток, где name_branch - ветка, которую нужно слить в текущую
